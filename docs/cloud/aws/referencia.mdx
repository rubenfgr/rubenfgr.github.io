## Instalar AWS CLI en debian

```sh
sudo apt-get install python-pip
sudo apt-get install awscli
```

---

## Autenticarse

Agregar las variables de entorno:

- AWS_ACCESS_KEY_ID
- AWS_SECRET_ACCESS_KEY
- AWS_SESSION_TOKEN

```sh
export AWS_ACCESS_KEY_ID=AKI******Q3FWX******
export AWS_SECRET_ACCESS_KEY=******kpCand12HkRa******Mt8aMfp5W6******
```
