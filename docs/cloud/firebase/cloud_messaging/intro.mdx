## Firebase Cloud Messaging (FCM)

Solución de mensajería multiplataforma que te ptermite enviar mensajes de forma segura y sin costo.

## Características

- Envía mensajes de notificación o mensajes de datos
- Segmentación de mensajes
  - A dispositivos individuales
  - A grupos de dispositivos
  - A dispositivos subscritos a temas
- Envío de mensajes desde apps cliente

## ¿Cómo funciona?

Incluye dos componentes principales:

- Entorno de confianza como Cloud Functions para Firebase o un servidor de apps para generar, segmentar y enviar mensajes.
- App cliente de Apple, Android o Web

Se pueden enviar mensajes desde:

- SDK de Firebase Admin o protocolos de servidor de FCM, (por ejemplo, en un backend con NodeJS)
- Compositor de Notificaciones en la consola de Firebase. Permite crear campañas.
