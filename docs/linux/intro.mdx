# Introducción

## Shell. Comandos básicos

```zsh
# Limpiar la consola
clear
# Ver historial de comandos utilizados
history
# Lista el directorio actual
ls
# Tipo de comando
type ls
# Con parámetros (l = long list, s = size, h = human readable)
ls -lah
# Información de un comando de "shell"
help type
# Información de  un comando que "no es de shell"
ls --help
# Manual de un comando
man ls
# Para que es un comando
whatis ls
# Información detallada de un comando
info ls
# Buscar comandos por palabra clave
apropos directory
apropos "list directory"
# Partir un comando en varias líneas
ls \
-lah
Directory
# Concatenar comandos
clear ; ls
# Operador AND (&&)
dsfasfasdf && ls
# Operador OR (||)
lkajsdfasdf || ls
```

## Shell. Atajos

```sh
# Ir a inicio de línea
Control + a
# Ir al final de línea
Control + e
# Limpiar pantalla
Control + l
# Últimos comandos utilizados
Control + p
# Buscar comandos utilizados
Control + r
# Utilizar el último comando que machea (coincide)
!ls
# Moverse derecha/izquierda
Control + b # back
Control + f # forward
# Argumento último comando
Alt + .
$!
# Autocompletar comandos
Tab
# Cortar la línea (situado al inicio)
Control + k
# Pegar la línea
Control + y
# Copiar/pegar texto con ratón
Seleccionar texto + ruleta
# Convertir mayúsculas/minúsculas
Alt + l/u
# Intercambiar 2 últimos caracteres
Control + t
# Intercambiar 2 últimas palabras
Esc + t
```

## Shell. Alias

```sh
# Crear un Alias
alias clist='clear ; ls'
# Conecer Alias en la sesión
alias
```

## Shell. Is Hashed

Es una tabla de referencias a comandos para acceder a ellos mucho más rápido

```sh
# Mostrar tabla de hash de la sesión actual
hash
# Limpiar la tabla para la sesióna actual
hash -a
```
