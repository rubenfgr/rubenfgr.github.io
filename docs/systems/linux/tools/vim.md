## Atajos

```sh
I o Insert --> Modo edición INSERT
ESC --> Salir del modo actual
V --> Selecciona la linea actual y entra en el modo VISUAL
Y --> Copia la selección
P --> Pegar selección
X --> Corta la selección
:w --> Guarda lo que estamos haciendo
:q --> Salir
:qa --> Guardar y salirç
:e + "ruta" --> abre el archivo indicado
:split --> divide la pantalla verticalmente o horizontalmente para editar varios archivos a la vez
:Ex Explorador de archivos
:help Ayuda de Vim

:e ~/.vimrc Editar el archivo de configuración de vim
```
